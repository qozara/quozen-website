# 🌐 Quozen Website

This repository hosts the public-facing landing page and marketing site for **Quozen**, a decentralized expense-sharing platform. 

It is a project of the [Qozara](https://qozara.org) research lab.

## 🚀 Live Links
- **Website:** [https://quozen.qozara.org](https://quozen.qozara.org)
- **Application:** [https://app.quozen.qozara.org](https://app.quozen.qozara.org)
- **Core Repository:** [qozara/quozen](https://github.com/qozara/quozen)

## 🛠 Tech Stack
- **Current:** Single-file static HTML5 & CSS3 (Minimalist, zero-dependency).
- **Future:** Planned migration to [Astro](https://astro.build) for enhanced content management and performance.

## 📂 Structure
- `index.html`: The main landing page.
- `LICENSE`: Apache 2.0 License.
- `/assets`: (Planned) Images, icons, and branding assets.
